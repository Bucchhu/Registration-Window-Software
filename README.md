"**Registration-Window-Software**" 
 User Registration Software (Java + MySQL Desktop Application)

A role-based desktop user registration system built using **Java Swing** and **MySQL**.  
This application allows users to register, edit their information securely, and enables administrators to authenticate and manage all registered user records through a dashboard interface.

---

## Features

- User registration with validation
- Admin login authentication
- Admin dashboard to view all users
- Secure credential verification before editing user data
- Profile update system with change detection validation
- Prevents blank or duplicate updates
- Password visibility toggle option
- Structured multi-screen navigation system
- MySQL database integration using JDBC

---

## Tech Stack

- Java (Core Java)
- Java Swing (GUI)
- MySQL Database
- JDBC (Database Connectivity)
- Git & GitHub (Version Control)
